# 小马逻辑

一款精美的逻辑推理小游戏，在 6x6 的网格中找出所有小马的位置。

## 游戏规则

1. **每种颜色只能放置1匹小马**
2. **每行只能放置1匹小马**
3. **每列只能放置1匹小马**
4. **小马之间不能相邻**（包括斜对角）

## 操作说明

- **单击**：标记/取消该格子（画 ✕）
- **双击**：尝试在该位置放置小马
- **撤销**：撤销上一步操作
- **重开**：重新开始当前关卡
- **提示**：显示一个小马的可能位置

## 运行方式

直接在浏览器中打开 `index.html` 文件即可开始游戏。

## 技术特点

- 纯原生 HTML/CSS/JavaScript 实现，无需任何依赖
- 响应式设计，适配移动端和桌面端
- 使用 CSS 动画提供流畅的交互体验
- 自动保存游戏进度到本地存储
- 算法生成可解的逻辑谜题

## 游戏特性

- 25 个关卡，越往后难度越高
- 连胜记录追踪
- 胜利动画和彩带效果
- 本地存储进度保存
